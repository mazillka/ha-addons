# Home Assistant Add-ons by Mazillka

A collection of custom add-ons for Home Assistant, designed to enhance your smart home experience with utility and accessibility tools.

## Add-ons Included

| Add-on                                                 | Description                                                                                                       |
| :----------------------------------------------------- | :---------------------------------------------------------------------------------------------------------------- |
| **[HomeReader TTS](./ha-homereader-tts-addon)**        | A beautiful, browser-based text-to-speech tool with adjustable voice settings and privacy-first local processing. |
| **[HA Download Manager](./ha-download-manager-addon)** | A powerful download management utility integrated directly into Home Assistant.                                   |

## Installation

### Easy Installation

Click the button below to add this repository to your Home Assistant instance:

[![Open your Home Assistant instance and show the add add-on repository dialog with a specific repository URL pre-filled.](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2Fmazillka%2Fha-addons)

### Manual Installation

1. Copy the URL of this repository: `https://github.com/mazillka/ha-addons`
2. In your Home Assistant UI, navigate to **Settings** > **Add-ons** > **Add-on Store**.
3. Click the three dots in the top right corner and select **Repositories**.
4. Paste the repository URL and click **Add**.
5. Once the repository is added, you can find the add-ons in the store under the **HA Add-ons by Mazillka** section.
